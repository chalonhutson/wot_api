Documentation

How to use our RESTful API...

First things first. If you don't have one, get an API token.

Please provide your Discord name. We don't have login functionality yet, so please keep it safe.

Use this API token as a query for all your requests. Be sure to add it at the end or you will get an error. Like so.

base-url/your_request?token=put-your-token-here

Base url

All RESTful api requests are through this base url...

https://wheel-of-time-api.herokuapp.com/api

Response

The response will come back as JSON data.

Get list of WOT characters. Will return a list of JSON objects, all of which have an id, name, title, race, and homeland.

/characters

Get an individual character. Will require the id as an argument. The id for each character is an integer.

/characters/~id~
